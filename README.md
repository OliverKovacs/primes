# primes

Simple prime sieve implemented in multiple languages.

### C
```bash
make c
```

### C++
```bash
make cpp
```

### Julia
```bash
make julia
```

### JavaScript
```bash
make node
```
```bash
make deno
```

### TypeScript
```bash
make ts_node
```
```bash
make ts_deno
```

### Python
```bash
make python
```

### Rust
```bash
make rust
```
